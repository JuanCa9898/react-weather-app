# React Weather App

A full featured weather application with React

## Live Demo

https://react-weather-app-seven-vert.vercel.app

## Features

- Search button
- Search weather by pressing enter
- Shows country, temperature, wind speed, wind direction and code
- Changes background in relation with the weather code of the city

## Technologies

- React
- JavaScript
- CSS
- Vite
